How to run GOOD by this example?
1. Copy the whole directory 'data' to a specific place of your computer, i.e., 'C:' on Windows;
2. Enter the 'data' directory in your computer through terminal;
3. Type the command 'python run_GOOD.py -dir_main C:\data -time 2022 32 3 -ftp whu -obs daily igs site_igs.list 0 24' for 
IGS observation downloading according to the file of 'site.list' site-by-site;
4. If you want to download more GNSS observations and products, please refer to help information in 
'run_GOOD.py' or the user mannual.
